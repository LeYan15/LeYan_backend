# Хакатон.Лента

## Описание проекта
Интерфейс прогноза спроса на 14 дней для товаров собственного производства с использованием гранулярности ТК-SKU-День.

## Стек технологий

- Python 3.11
- Docker
- Django DRF
- swagger API
- Unitest
- js
